Procedimento de Instalação do NeuroSis

  • Ter um dominio disponivel (ex: www.lerolero.com.br);
  • Ter um servidor de hospedagem com as especificações tecnicas abaixo (obrigatoriamente)
    • Sistema Operacional Linux
    • Servidor APACHE
    • PHP 5.6 (Não pode ser PHP 7)
    • Banco de Dados MariaDB

  • Configurar os endereços DNS para apontar para o servidor de hospedagem

  • Criar um novo banco de dados (Obrigatorio codificação padrao UTF8)
    importar o arquivo sql inicial na tabela

  • Alterar as configurações do arquivos config.php

    • Endereço do domínio

    • define(‘URL’, ‘http://lerolero.com.br‘);

    • Tipo do Banco de dados

    • define(‘DB_TYPE’, ‘mysql’);
    • Endereço do banco de Dados
    • define(‘DB_HOST’, ‘127.0.0.1’);
    • Nome do Banco de Dados
    • define(‘DB_NAME’, ‘NeuroSis’);
    • Usuario do Banco de Dados
    • define(‘DB_USER’, ‘root’);
    • Senha do Banco de Dados

    • define(‘DB_PASS’, ‘lerolero’);

    • Nome da Aplicação

    • define(‘APP_NAME’, ‘NeuroSis’);

  • Subir todos os arquivos para o servidor.
